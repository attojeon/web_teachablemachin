# 고양이/강아지 구분하는 AI
- 30000만장의 사진으로 티처블머신으로 훈련시킴.
- 왠만한 고양이/강아지 구분함. 


## 설치법 및 사용법 
1. VSCode에서 'Go Live'를 사용하여 로컬웹서비스로 사용할 수 있음. 
2. netlify, cloudtype 등 공짜로 웹서비스 할 수 있는 서버에 파일과 my_model 을 통째로 올리기
3. 깃허브의 앱서비스를 사용해 볼 것
